Project 1
Ksimul8 - Kubernetes Scheduler Simulator

Python-based simulation application for the Kubernetes scheduler. The project mainly focuses on the scheduler used by Kubernetes which is called the Kube-scheduler. The problem arises when testing the behavior of the Kube-scheduler in a real cluster gets complicated and costly. Therefore, we decided to create a simulator to test the behavior of the Kube-scheduler under different scenarios.

Project 2
My Message in The Bottle

A Web-based messaging application in which the user (sender) can decide the date and time of message delivery to the recipient. The project was implemented using a combination of Python libraries such as flask (to implement the API), Celery to execute periodic asynchronous tasks, and Docker-compose to run the containers with all the processes needed.
